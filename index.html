Regex Syntax Guide

  • a — single character
  • ab — concatenation
  • a|b — union (a or b)
  • a* — zero or more (Kleene star)
  • a+ — one or more (Kleene plus)
  • a? — optional (zero or one)
  • (a|b) — grouping

Language Notation Guide

Exponent Notation

  • a^n b^n — n a's followed by n b's
  • a^2n b^n — 2n a's followed by n b's (coefficients)
  • a^n b^m — independent counts (different variables)
  • Constraints: n >= 1 or n >= 0 (default)

Word Notation

  • wcw^R — string w, then c, then w reversed
  • ww^R — string w then w reversed
  • w | #a = #b — equal count of a and b

PDA Details

  • Transitions: input, pop / push
  • Initial stack symbol: z0
